# Mobile audit — v1.0.1

The full site shell and every component category were reviewed at 320, 360, 390, 430, 768 and 1024 CSS pixels.

Changes include:
- compact mobile header and off-canvas navigation
- fluid global search and fixed mobile results panel
- single-column library and documentation layouts
- scroll-safe category filters, tables and Kanban boards
- fluid previews, forms, cards, charts and templates
- reduced mobile typography, spacing and radii
- stacked actions and full-width touch targets
- mobile-safe modals, drawers, code blocks and API tables
- special handling for 320–360 px devices
